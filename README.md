# Projeto Django - Faculdade

Este projeto foi desenvolvido como atividade prática da disciplina, utilizando o framework Django.

## 🚀 Tecnologias utilizadas
- Python
- Django

## 📁 Estrutura do projeto

- `meu_projeto/` → projeto principal
- `meu_aplicativo/` → aplicação criada

## ⚙️ Como executar o projeto

1. Criar ambiente virtual:
```bash
python -m venv venv
2. Ativar o ambiente:

venv\Scripts\activate

3. Instalar dependências:
pip install django

4. Aplicar migrações:
python manage.py migrate

5. Rodar o servidor:
python manage.py runserver
